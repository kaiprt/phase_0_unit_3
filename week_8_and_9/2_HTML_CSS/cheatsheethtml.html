Variable Scope

Definition

Variable scope defines where in a program a variable is accessible.

Symbol

Scope
$

Global Variable
@

Instance Variable

[a-z] or _

Local Variable
[A-Z]

Constant
@@

Class Variable

A useful technique to find out the scope of a variable is to use the defined? method.

x = 10
=> 10
defined? x
=>"local variable"

Scopes: Broken Down

  • Local Variables- local to the code construct in which they are declared.
  • Global Variables- accessible from anywhere in the Ruby program, regardless of where they are declared.
  • Class Variables- a variable that is shared amongst all instances of a class.
  • Instance Variables- similar to Class variables except that their values are local to specific instances of an object.
  • Constant Scope- values which, once assigned a value, should not be changed.

Data Types

Definition

A data type is a set of values, and the allowable operations on those values. Ruby has six data types:

  • Number
  • String
  • Array
  • Hash
  • Boolean
  • Symbol

Numbers

Types of numbers used in the Ruby Programming Language:

  • 123 (Integer)
  • 123.45 (Float)
  • 1.2e-3 (Scientific Notation)
  • 0xffff (Hex)
  • 0b01011 (Binary)
  • 0377 (Octal)

Strings

A string is a data type representing textual data in computer programs. Strings are characters enclosed in double or single quotes.

'I'm a string'
"Me too!"

Arrays

An array is an ordered group of objects. To access an element in the array you must call it by it's index (starts at 0).

my_array = [1,2,3,4,5]
#=> nil
my_array[2]
#=> 3

Hashes

Similar to arrays, hashes store a collection of values in a key and value fashion. Hashes use a value you provide as the key.

Hash["FirstName" => "Mark", "LastName" => "Brown", "Status" => "Bacon"]
Key = "FirstName", "LastName", "Status"
Value = "Mark", "Brown", "Bacon"

Booleans

Boolean values represent yes or no (true/false) and are based off of pure logic.

3 == 4
#=> false
3 == 3
#=> true
x = 5; x > 2
#=> true

Operators

Arithmetic Operators

Operator

Description

Example

+

Addition - Adds values on either side of the operator

a + b will give 30

-

Subtraction - Subtracts right hand operand from left hand operand

a - b will give -10

*

Multiplication - Multiplies values on either side of the operator

a * b will give 200

/

Division - Divides left hand operand by right hand operand

b / a will give 2

%

Modulus - Divides left hand operand by right hand operand and returns remainder

b % a will give 0

**

Exponent - Performs exponential (power) calculation on operators

a**b will give 10 to the power 20

Comparison Operators

Operator

Description

Example

==

Checks if the value of two operands are equal or not, if yes then condition becomes true.

 15 == 10
#=> false

!=

Checks if the value of two operands are equal or not, if values are not equal then condition becomes true.

 15 != 10
#=> true

>

Checks if the value of left operand is greater than the value of right operand, if yes then condition becomes true.

 4 > 2
#=> true

<

Checks if the value of left operand is less than the value of right operand, if yes then condition becomes true.

 4 < 2
#=> false

<=

Checks if the value of left operand is greater than or equal to the value of right operand, if yes then condition becomes true.

 4 >= 4
#=> true

<=

Checks if the value of left operand is less than or equal to the value of right operand, if yes then condition becomes true.

 2 <= 3
#=> true

<=>

Combined comparison operator. Returns 0 if first operand equals second, 1 if first operand is greater than the second and -1 if first operand is less than the second.

 4 <==> 2
#=> 1

===

Used to test equality within a when clause of a case statement.

 (1...4) === 2
#=> true

Assignment Operators

Operator

Description

Example

=

Simple assignment operator, Assigns values from right side operands to left side operand

c = a + b will assigne value of a + b into c

+=

Add AND assignment operator, It adds right operand to the left operand and assign the result to left operand

c += a is equivalent to c = c + a

-=

Subtract AND assignment operator

c -= a is equivalent to c = c - a

*=

Multiply AND assignment operator

c *= a is equivalent to c = c * a

/=

Divide AND assignment operator

c /= a is equivalent to c = c / a

%=

Modulus AND assignment operator

c /= a is equivalent to c = c / a

**=

Exponent AND assignment operator

c **= a is equivalent to c = c ** a

Ruby Classes

Definition

Because Ruby is an object-oriented programming language, defining a Ruby class essentially creates a manual for how a particular object operates. Dogs bark, cars drive, and planes fly - all of these are perfect examples of how Ruby classes operate. Each method created within the class guides what the object does (ie. bark, drive, fly).
